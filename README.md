# happy-nations
An interactive simulation tool that empowers policymakers to test how socio-economic interventions predictably impact a nation's collective well-being.

## Project Description
Traditional governance relies heavily on Gross Domestic Product (GDP) to measure national progress, often ignoring the human cost of poor social structures. This project bridges data science and public policy by transforming the World Happiness Report into an interactive simulation engine.

Using Python and Streamlit, the application models the historical relationship between happiness and its six underlying pillars (Economy, Social Support, Health, Freedom, Generosity, and Trust). Users can select any nation, manipulate sliders to mimic real-world policy shifts (e.g., “What if we invest heavily in community infrastructure to raise Social Support by 15%?”), and instantly observe the predicted impact on that country's global happiness rank and score.
